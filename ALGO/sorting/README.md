# Sorting Algorithms

## dc__merge_sort

* Worst-case: `O(nlog(n))`
* Average-case: `O(nlog(n))`
* Best-case: `O(nlog(n))`
* Category: dc

## dc__quick_sort

* Best-case: `O(nlog(n))`
* Average-case: `O(nlog(n))`
* Worst-case: `O(n^2)` when already sorted with opposite `condition`
* Category: dc

## insertion_sort

* Best-case: `O(n)`
* Average-case: `O(n^2)`
* Worst-case: `O(n^2)`
* Category: ms
